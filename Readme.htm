WarpCD V0.2.0

WarpCD is a CD-player with skin support. Use any of the avaiable WinAmp skins to change it's appearance. This player uses the IOCtrl-interface and therefore does not need MMOS2. This program is released under the Gnu Public Licence (GPL). See below.

Usage

On first start WarpCD defaults to the first CD-Rom drive of your system. Use the options menue item (right mouse button) to change that. You may change the default skin directory, too. Compressed skins are not supported with this release.

How to compile

You need the GNU file utilities, gnu make 3.76.1 and the EMX development environment. Install the sources in the WarpCD directory (the default path). Cd to the source directory and type make.

For the release package you also need WarpIn the Warp installer. Cd to the source directory and type make distribution.
